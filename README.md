# QuizPdf
[Notion2QuizPdf](https://github.com/kyasuda516/notion2quiz-pdf)によって書き出されたPDFの集まりです。

Notion2QuizPdfを用いると、Notion記事を問題集のようにし、結果的にこのリポジトリにあるようなPDFを作成します。
